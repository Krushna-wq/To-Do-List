# To-Do-List
A modern and stylish To-Do List UI built using **HTML** and **CSS**. 
This is a front-end design only (not yet functional with JavaScript).

## ✨ Features

- Clean, modern UI
- Beautiful color palette
- Responsive layout
- Font Awesome icons
- Clear empty state message

## 🚀 Tech Used

- HTML5
- CSS3
- Font Awesome (via CDN)
